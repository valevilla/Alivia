# Alivia: Sistema de Sufragio Descentralizado con Asistencia de IA

**Alivia** es una plataforma de votación electrónica de alta seguridad diseñada para eliminar las barreras tecnológicas y garantizar la transparencia absoluta en procesos electorales en América Latina. El sistema combina la inmutabilidad de la tecnología **Blockchain** con el acompañamiento humano de un **Agente Autónomo (Qawi)**.



## 🌟 Características Destacadas

- **Agente Autónomo Qawi:** Una interfaz asistida que utiliza lenguaje natural y guías visuales para ayudar a personas con dificultades técnicas o adultos mayores a completar su voto sin errores y transparente. La alarma SOS tiene integrada.
- **Ruta Segura con Google Maps:** Integración de GPS que traza el camino más seguro hacia el local de votación, monitoreando incidentes o zonas de riesgo en tiempo real.
- **Token de Sesión Único (MFA Dinámico):** Sistema de autenticación que genera una clave única por sesión, blindando el acceso contra ataques de phishing o robo de identidad.
- **Trazabilidad Incorruptible:** Cada voto se registra como una transacción en una red blockchain, generando un recibo digital (Hash) que permite al ciudadano verificar que su voto fue contado sin revelar su elección. Se tiene un dashboard del mapa del país de votación para determinar en tiempo real cuentas personas votaron y hacer comparativa con resultados de voto oficial.
- **Alarma SOS:** Notificar mediante aplicativo irregularidades antes y durate votaciones hasta finalizar los resultados oficiales.

## 🛠️ Stack Tecnológico

- **Frontend:** React.js + Tailwind CSS
- **Iconografía:** Lucide React
- **Seguridad:** Arquitectura de Agentes Autónomos (Reglas de Supervivencia)
- **Criptografía:** Simulación de hashing SHA-256 para integridad de datos.
- **Dashboard:** PowerBI y Google map.
